📘 PART 1. 준비 및 기본기

1. 기본 환경 설정 (작업 전 필수)

환경 설정 창 열기 Ctrl + K

  • 초기화: 하단의 [Reset Preferences] 클릭 → OK → 재시작 (작업 이력만 지워지고 파일은 안전합니다)
  • 시작 화면 끄기: General → [Show the Home Screen...] 체크 해제
  • 폰트 이름 한글로 보기: Type → [Show Font Names in English] 체크 해제
눈 피로도 감소

User Interface → Brightness를 어두운 테마로 설정

자동 저장 (매 중요)

File Handling → Automatically Save... 체크 후 1~2분 간격로 설정

작업 화면(패널) 및 툴바 정리

  • 상단 Window → Workspace → Essentials Classic 선택
    (패널이 꼬였을 때는 Reset Essentials Classic 클릭)
  • 선택할 때마다 뜨는 성가신 바 끄기: 상단 Window → Contextual Task Bar 체크 해제

2. 화면 조작 및 새 문서 설정

새 문서 만들기 Ctrl + N

※ 주의: 크기 숫자를 적기 전에 반드시 '단위(mm)'부터 맞추세요!

  • 크기: 500 x 500 mm (정사각형 로고 기준) / A4 사이즈
  • 컬러 모드: 웹(모니터)용은 RGB / 인쇄용은 CMYK
  • 해상도(Raster Effects): 웹/일반 작업은 72ppi (크면 컴퓨터가 느려집니다)

기본 선택 / 그리기

  • V 선택 도구 (검은 화살표)
  • A 직접 선택 (점 수정 / 흰 화살표)
  • M 사각형 도구
  • L 원형 도구
  • Shift+O 아트보드(대지) 도구

화면 제어 / 속성 잠금

  • Space + 드래그 화면 이동
  • Alt + 휠 화면 확대 / 축소
  • Ctrl+2 개체 잠금 (해제: Ctrl+Alt+2)
  • 더블클릭 아이솔레이션(수정) 모드 나가기: ESC
  • D 기본 색상 (흰 면, 검은 선)
  • / 색상 채우기 없음 (투명)

오브젝트 잠금(Lock) 및 숨김(Hide)

위치: 메뉴바 - Object (오브젝트가 선택되어 있으면 활성화 됨)

👁️ 숨김 (Hide)
  • 숨김 (Hide): Ctrl + 3
  • 숨김 해제 (Show all): Ctrl + Alt + 3

🛠️ PART 2. 핵심 기능 사전

💡 실무를 위한 추가 핵심 기능 모음

📄 문서 및 대지(Artboard)

  • 도련 수정: 문서 제작 시 입력하거나, File → Document Setup에서 도련 재설정
  • PDF 추출: File → Export → Export for Screens
    - Single File: 한 파일에 여러 아트보드 포함
    - Multiple Files: 아트보드 1개당 1개의 PDF 생성
  • 대지 사이즈 측정: Artboard 툴(Shift+O) 선택 후 컨트롤 패널(상단) W/H 구간 확인
  • 대지 영역만 보기: View → Trim View

✒️ 패스(Path) & 앵커 포인트

  • 도형 생성: Alt+클릭 후 수치 입력 (클릭 지점이 도형의 중앙이 됨)
  • Pencil Tool(N): Enter 눌러 옵션. Fidelity(Smooth쪽일수록 부드러운 패스 생성), Keep selected(선택 유지), Alt key toggles(Alt 누를 시 Smooth Tool로 전환), Edit selected path(선택된 패스에 덧대어 수정)
  • Pen Tool: 곡면 예상 앵커 찍기 → Anchor Point Tool 활용 곡면 생성 (한쪽 핸들만 조절 가능).
    - Pen + Alt: 곡면 생성 도구 토글 / Pen + Ctrl: Direct Selection Tool 토글
  • 앵커 포인트: Object → Path → Add Anchor Point (도형에 앵커 추가) / 떨어져 있는 앵커 모두 선택 후 우클릭 → Average → Both (앵커 모으기)
  • 가위(C) & 칼(Knife) 도구: Scissor(열린 패스로 마무리), Knife(닫힌 패스로 잘라냄, 선택하지 않아도 적용 가능)

🎨 색상, 그라디언트 & 라이브 페인트

  • Swatches (견본): Global 체크 시 사용된 컬러 일괄 변경 가능 / 면색이 활성화된 상태에서 + 눌러 등록
  • Gradient (Freeform): 원하는 지점 클릭해 컬러 스탑 추가. (삭제: 클릭 후 Delete, 이동: 드래그, 색상변경: 더블클릭)
    - 스와치 등록: 그라디언트 패널 색상박스 ▼ → 좌측 하단 다운로드 모양 클릭
  • Live Paint Tool(K): 열린 패스도 닫힌 패스처럼 색칠 가능. 사용 전 모든 패스 선택. 면색(클릭), 선색(Shift+클릭), 스포이드(Alt).
    - 적용 시 기존 선 속성(brush 등)이 사라지므로 Expand 후 설정하거나 칠할 면을 뒤에 따로 둠. (해제: Object→Live Paint→Release / 굳히기: Expand)
  • Mesh(U): 부드럽게 혼합 및 입체 왜곡 (Lasso Tool과 자주 사용)
  • 같은 색 도형만 선택: Select → Same → Fill Color

✨ 변형(Transform), 효과 & 패턴

  • Effect & 중첩: Zigzag(Points를 Smooth로 지정 시 물결), Blur 등.
    - Appearance(모양) 패널에서 적용된 효과 이름 선택해서 수정. 효과 이중 사용 시 'Apply new effect' 안내창 발생.
    - 효과 적용 후 Expand Appearance 필수.
  • Blend(Ctrl+Alt+B): 레이어 순서(위:시작→아래:끝) 영향 받음. 해제 시 중간 경로가 투명 선으로 남으므로 지우고 재적용.
  • 패턴(Pattern) 등록: Swatches로 드래그 또는 Object → Pattern → Make.
    - 모양 변경: Transform 기능(단, Transform Objects 체크 해제 후 사용). 배경색: Appearance 활용.
  • 3D: Classic(모양 확장/면 분리 가능) vs 3D(모양 확장 불가, 이미지로 굳어버림)

📏 정렬, 격자 및 특수 도구

  • Distribute Spacing (간격 정렬):
    - Key Object 미설정: 첫 도형과 끝 도형 기준 중간 간격 자동 분배
    - Key Object 설정: 수치 입력 가능 (Enter 안됨, 세로/가로 간격 버튼 눌러 적용)
  • Align 활용 앵커 모으기: 마지막으로 선택한 점을 기준으로 정렬됨. Key Anchor 설정 후 가운데 → 좌/우 정렬 수행.
  • Compound Path(Ctrl+8): 여러 개의 경로를 완벽히 하나의 경로처럼 인식. (단순 묶기인 Group과의 차이점)
  • 아이소메트릭 그리드 (Isometric Grid):
    - 정육각형 기반, Guides 역할을 할 선의 개수는 홀수 (예: 101개).
    - 긴 선 생성 → Transform(간격 14px, 100개 복제) → Expand → Rotate(60도) → Reflect(Vertical).
  • Width Tool(Shift+W) & Stroke Profile: 선의 특정 폭 도구 및 전반적인 선의 형태 변경.
  • Blending (혼합 모드): Transparency 패널(Opacity 왼쪽 Normal 부분)에서 변경.

일정한 크기로 확대/축소 (Offset Path)

원의 반지름 등을 기준으로 정확히 커지거나 작아집니다.

Object → Path → Offset Path

★ 꿀팁: 사칙연산 입력 지원!
직접 계산할 필요 없이 창에 `(40-33.6)/2` 라고 입력하면 자동으로 반영됩니다.

4. 형태를 바꾸는 고급 도형 & 특수 도구

도형 자르고 합치기 (Shape Builder) Shift + M

※ 작업할 도형들을 '모두 선택'하는 것이 1순위입니다!

  1. 분리: 그냥 클릭
  2. 합치기: 훑듯이 클릭 + 드래그
  3. 지우기(빼기): Alt 누른 상태로 클릭 / 드래그

⚠️ 선이 완벽히 맞물리지 않으면 잘리지 않습니다! 억지로 겹치게 늘린 후 자르세요.

정확한 맞물림을 보려면 Ctrl + Y (윤곽선 모드)를 사용하세요.

패스파인더 (Pathfinder)

두 개 이상의 오브젝트를 합치거나 잘라 하나의 오브젝트로 만들어줍니다. (메뉴: Window → Pathfinder)

Shape Modes

두개 이상의 오브젝트를 합치거나 잘라 하나의 오브젝트로 만들어줌

  • 1. Unite: 합치기 (가장 앞에 있는 오브젝트의 색상이 적용됨)
  • 2. Minus Front: 앞면 오브젝트 제외 (선택한 오브젝트 앞에 있는 오브젝트 영역 삭제)
  • 3. Intersect: 교차영역 남기기 (선택한 오브젝트 중 겹친 부분만 남김)
  • 4. Exclude: 교차영역 제외 (교차 부분만 삭제, 앞에 있는 옵젝의 색상 적용)

Pathfinder

※ 작업 후 그룹화 되기 때문에 별도 작업 하려면 그룹 해제 후 사용

  • 1. Divide: 나누기 (겹치는 패스를 기준으로 옵젝 분리)
  • 2. Trim: 오브젝트 분리 (앞 오브젝트와 겹치는 영역 삭제 후 분리)
  • 3. Merge: 병합 (오브젝트 색이 같으면 합치고 다르면 분리)
  • 4. Crop: 자르기 (겹치는 부분과 맨 앞쪽 오브젝트의 패스만 남김 - 분리도 동시에)
  • 5. Outline: 겹치는 오브젝트 모두 잘리고 패스는 선으로 남음
  • 6. Minus Back: 맨앞 오브젝트를 기준으로 후면 오브젝트 영역만큼 제거

단순 묶기: 그룹 (Group)

도형을 한 묶음처럼 취급. 각각의 개체는 살아있습니다.

💡 [실무 참고] 로고의 6가지 종류와 활용 가이드

브랜드 아이덴티티(BI) 기획 시, 브랜드의 목적과 성격에 맞는 로고 타입을 선택하는 것이 매우 중요합니다.

1. 심볼형 (Symbol)

상징적인 아이콘(그림)과 폰트가 함께 사용되는 형태 (ex. 애플, 나이키)

  • 장점: 가독성이 높고 강하게 기억에 남으며 스토리텔링이 용이함
  • 추천: 웹/모바일/오프라인 등 다양한 매체에 변형 활용해야 하는 브랜드

2. 워드마크형 (Wordmark)

글자 자체에 디자인 요소가 삽입되어 마크가 되는 형태 (ex. 넷플릭스, 반스)

  • 장점: 매우 직관적이며 브랜드 이름을 빠르게 인식시킬 수 있음
  • 추천: 빠른 인지도 확장이 필요한 신생 브랜드나 흔한 단어로 이루어진 브랜드

3. 텍스트형 (Text)

디자인 요소 없이 서체의 폰트(자간, 행간 등)로만 표현 (ex. 헤라, 탬버린즈)

  • 장점: 서체에 따라 감성적이고 세련된 무드 연출에 매우 적합함
  • 추천: 감성 카페, 쇼핑몰, 방송 타이틀 등 무드와 분위기가 중요한 곳

4. 캘리그라피형 (Calligraphy)

손으로 직접 쓴 붓글씨, 핸드라이팅 형태 (ex. 설빙, 옥이네 김밥)

  • 장점: 전통, 오랜 경력, 기품 있고 동양적인 고급스러움 표현 가능
  • 추천: 타겟 고객 연령층이 높거나 전통과 경험을 브랜드 자산으로 삼는 요식업

5. 캐릭터형 (Character)

사람이나 동물을 캐릭터로 표현한 시그니처 형태 (ex. 빽다방, 노티드)

  • 장점: 거부감이 적고 친근함을 주며 굿즈 등 다채로운 시각적 베리에이션 가능
  • 추천: 스토리텔링이 중요하거나 고객(주로 젊은 층/키즈)과 적극적 소통이 필요한 브랜드

6. 엠블럼형 (Emblem)

프레임 안에 폰트, 심볼, 슬로건 등이 모두 포함된 덩어리 형태 (ex. 스타벅스, 버거킹)

  • 장점: 무게감이 있어 웅장함, 위엄, 정통성, 전문적인 분위기 시각적 전달에 탁월함
  • 추천: 대학교, 정부기관 또는 오랜 역사와 전통을 어필해야 하는 조직

3. 건물 오른쪽 벽면에 그림자가 생기는 곳 모두 추가 작업
* plus 물체가 빛을 가려서 생기는 어두운 부분이 어디일지 고민하면서 작업할 것

4. 건물 왼쪽 벽면 → 나무가 튀어나온 부분을 고려하여 그림자가 생기는 부분을 모두 검은색 면으로 그리기

9. 블렌드 도구를 이용한 일러스트 실전 제작

💡 블렌드(Blend) 란?

블렌드 도구를 사용하여 두 개체 사이의 모양과 색상을 자연스럽게 변화시키며, 라이노컷(Linocut)과 같은 판화 느낌 등 다양한 디자인 효과를 만들 수 있습니다.

🚀 스텝별 제작 과정

Step 1. 블렌드를 활용하여 라이노컷 작업하기

1. [File]-[Open]으로 'Style14_1_blend_start.ai' 파일을 엽니다. 물고기 스케치 그림이 준비되어 있습니다.

2. 면색 없이 선색만 검은색으로 설정합니다. 화면을 확대하고 Pen Tool(펜 도구)을 선택합니다. 블렌드를 만들기 위해 라인을 2개 그립니다. Ctrl+F10을 눌러 [Stroke] 패널을 열고 Width Profile을 첫 번째 모양으로 바꿉니다.

3. 두 개체를 모두 선택하고 블렌드 실행 명령인 Ctrl+Alt+B를 누릅니다. 기본 블렌드가 적용되며 가운데 라인이 추가로 생성됩니다.

Step 2. 물고기 몸통 부분 블렌드 만들고 조절하기

1. Pen Tool로 몸통 부분에 라인을 3개 그립니다. 모두 선택하고 Ctrl+Alt+B를 누릅니다. Blend Tool을 더블클릭하여 Steps로 10을 입력하고 OK를 누릅니다.

2. Width Tool로 라인의 두께 조절합니다. 위쪽은 두껍게, 아래쪽은 얇아지도록 섬세하게 작업합니다.
* plus 다듬어야 할 부분이 있다면 점의 위치와 핸들을 조절하여 자연스럽게 만들어줍니다.

3. 끝부분의 점 3개를 같은 위치로 이동합니다.
* Tip: 처음부터 같은 위치에 그리면 선이 연결되어 블렌드가 만들어지지 않기 때문에 떨어지게 그린 뒤 나중에 위치를 옮깁니다.

4. 몸통 개체를 선택합니다. [Object]-[Blend]-[Expand]로 확장합니다.
* Tip: Expand Appearance는 선까지 모두 확장합니다. 따라서 블렌드 작업을 할 때에는 Expand를 적용해야 선인 상태가 유지됩니다.

5. Direct Selection Tool로 겹치는 부분의 점을 하나씩 이동하여 모양을 다듬어줍니다.

Step 3. 물고기 배경 베이스 만들기

물고기의 머리-몸통 부분과 지느러미가 겹치는 부분이 지저분하게 만들어질 수 있으므로 베이스 개체를 만들어 보겠습니다.

1. Ctrl+A로 전체 선택한 후 Ctrl+2를 눌러 개체를 잠급니다. Pen Tool로 머리와 몸통 부분을 둘러싸듯이 외곽을 그려줍니다.

2. 시작점으로 돌아와 막힌 개체로 만들고 면색을 지정합니다. Ctrl+Shift+[를 눌러 맨 뒤로 배열합니다.

3. F7을 눌러 [Layers] 패널을 엽니다. 작업하던 Layer 2를 잠그고 새로운 레이어를 만듭니다. Layer 2의 눈을 끕니다.

4. 완만한 곡선으로 지느러미를 모두 그립니다. 꼬리지느러미는 한 개만 그립니다.
* plus 펜 도구로 곡선을 유려하게 만들 수 있어야 합니다.

5. Ctrl+A로 전체 선택한 후 Ctrl+C로 복사하고 바로 Ctrl+2로 모든 개체를 잠급니다. Ctrl+F로 제자리에 붙여넣기 한 뒤 면색 없이 선색을 검은색으로 지정합니다.

Step 4. 지느러미 블렌드로 표현하기

1. Scissors Tool(가위 도구)로 지느러미 위와 아래의 뾰족한 점 두 곳을 클릭하여 패스를 끊습니다.

2. [Stroke] 패널에서 Width Profile 1로 설정하고 Ctrl+Alt+B로 블렌드를 적용합니다.

3. Blend Tool을 더블클릭하여 Steps로 18을 입력하고 OK를 누릅니다.

4. 나머지 지느러미도 모두 같은 방법으로 작업합니다. 크기에 따라 블렌드의 개수를 다르게 하거나, Width Tool로 선 두께를 임의로 조절합니다. Ctrl+Alt+2로 잠긴 개체를 해제합니다.

5. 꼬리 쪽 개체를 모두 선택하고 복사합니다. Reflect Tool(반사 도구)을 더블클릭하여 Horizontal을 선택하고 OK를 누릅니다.

6. Ctrl+Shift+[로 맨 뒤로 배열합니다. 스케치를 무시하고 그림과 같이 배치합니다. [Layers] 패널에서 Layer 2를 가장 위로 이동하고 잠금을 해제합니다.

Step 5. 눈과 수염 표현하기

1. Layer 2와 Layer 3의 눈을 끄고, 새로운 레이어를 만들어 가장 위로 이동합니다. 눈 개체의 가장 큰 외곽을 막힌 개체로 만들고 면색에 기본 그라디언트를 적용합니다.

2. Gradient Tool(그라디언트 도구)을 클릭하고 원의 왼쪽 색상점을 K20, 오른쪽 색상점을 K5로 입력합니다.

3. 개체에 검은색 선을 적용합니다. Width Tool로 일부는 두껍게 일부는 얇게 조절합니다.

4. 타원을 회전시켜 눈 위에 올립니다. 안쪽은 진한 갈색, 바깥쪽은 연한 갈색으로 방사형 그라디언트를 적용하고 눈 모양에 맞게 간격을 조절합니다.

5. 검은색 타원은 눈 안쪽에 배치하고, 그 위에 작은 흰색 타원을 그려 눈의 반짝거리는 부분을 표현합니다.

6. 하나의 라인으로 수염 부분을 그립니다. 두께는 3pt, Cap 모양은 둥글게 처리합니다.

7. Width Tool로 선 두께를 조절하여 그림과 유사하게 만들고 [Object]-[Expand Appearance]를 눌러 면으로 확장합니다.

Step 6. 색상 표현하기

1. [Layers] 패널을 엽니다. 'Layer 1'을 삭제하고 경고창이 뜨면 Yes를 누릅니다. 'Layer 2'를 선택하고 Ctrl+Alt+2를 눌러 잠긴 개체를 해제합니다.

2. Selection Tool(선택 도구)로 몸통의 노란색 개체를 더블클릭합니다. 해당 개체만 작업할 수 있는 Isolation 모드로 바뀝니다. 개체를 선택하고 Ctrl+F9를 눌러 [Gradient] 패널을 열고 Freeform Gradient를 선택합니다.
* plus 여기까지의 과정이 'Style14_1_blend_start_course1.ai'으로 저장되어 있습니다.

3. 머리와 옆구리 쪽에 C0, M12, Y100, K0, 가운데 몸통에는 C0, M80, Y100, K0, 마지막 꼬리 쪽에는 C5, M90, Y100, K5를 지정합니다. Selection Tool로 작업 화면의 빈 곳을 더블클릭하면 Isolation 모드가 해제됩니다.

4. 지느러미 부분을 확대하여 아래쪽 면 개체를 더블클릭합니다. 선형 그라디언트를 선택하고 양쪽 색상은 C0, M90, Y100, K0, 오른쪽은 Opacity를 0%로 지정합니다. 그라디언트를 직접 드래그하여 방향을 설정합니다. 작업을 마치면 그라디언트를 [Swatches] 패널에 저장합니다.
* Tip: 그라디언트 패널 타일 섬네일의 목록 열기 → 'Add to Swatches' 아이콘 클릭 → Swatches 패널에 등록 확인

5. 다른 지느러미 부분도 동일하게 작업합니다. 양쪽 색상은 C0, M100, Y70, K0이며 오른쪽은 투명하게 처리합니다. 그라디언트 방향을 직접 설정하고 [Swatches] 패널에 저장합니다.

6. 다른 지느러미 부분도 동일하게 작업합니다. 양쪽 색상은 C20, M100, Y60, K0이며 오른쪽은 투명하게 처리합니다. 그라디언트 방향을 직접 설정하고 [Swatches] 패널에 저장합니다.

7. 반대쪽 지느러미도 저장했던 그라디언트들을 이용해 비슷하게 작업합니다. 꼬리지느러미를 확대하고 안쪽 면 개체를 더블클릭합니다.

8. 왼쪽은 C67, M0, Y34, K0, 가운데는 C90, M100, Y20, K30, 마지막은 C0, M100, Y100, K0으로 지정합니다. 그라디언트 방향을 직접 설정하고 [Swatches] 패널에 저장합니다.

9. 남은 꼬리지느러미도 그라디언트를 적용합니다. 가운데 적용된 남색은 삭제하고 왼쪽을 투명하게 처리합니다.

10. Ctrl+A로 모든 개체를 선택한 뒤 [Object]-[Blend]-[Expand]를 눌러 모든 블렌드 개체를 선으로 활성화합니다. 다시 [Object]-[Expand Appearance]를 눌러 면 개체로 확장합니다.

11. 검은 개체 부분을 선택하면 한 덩어리씩 그룹으로 되어 있습니다. Ctrl+8을 눌러 하나의 개체로 결합합니다. 개체가 투명해지면 견본 패널에 등록해 둔 그라디언트를 적용하고 방향을 직접 설정합니다.

12. 나머지도 동일하게 작업합니다. 먼저 컴파운드 패스로 결합하고 개체의 면색에 그라디언트나 단색을 적용합니다.

13. 몸통의 검은색 개체를 선택하고 Ctrl+8을 누릅니다. 그라디언트 첫 번째 색상점은 C0, M90, Y100, K0, 두 번째 색상점은 C90, M100, Y20, K30, 세 번째 색상점은 C0, M100, Y100, K0으로 지정하고 각도나 위치 등도 조정합니다.

14. 꼬리지느러미도 동일하게 작업합니다. 양쪽 색상은 C90, M100, Y20, K30이며 한쪽은 투명하게 처리합니다.

15. 물고기의 수염은 C0, M12, Y100, K0으로 지정합니다.

Step 7. 그림자와 물 질감 표현하기

1. [Layers] 패널을 열고 가장 하단에 새로운 레이어를 추가합니다. 문서 사이즈와 같은 크기의 사각형을 그린 뒤 면색을 임의로 설정합니다.

2. [Gradient] 패널에서 Freeform Gradient를 선택합니다. 색상점을 5개를 만듭니다. 가운데와 왼쪽 아래는 C90, M100, Y20, K30의 메인 색상을 사용합니다. 그 외의 색은 각각 C65, M100, Y35, K27과 C100, M80, Y34, K24와 C100, M100, Y30, K45를 지정합니다.

3. 'Layer 2'를 선택하고 잉어의 몸통 중 아래쪽 그라디언트 개체를 선택합니다. 복사 후 F7을 눌러 [Layers] 패널을 엽니다. 밑에서 두 번째 위치에 새로운 레이어를 만듭니다.
* plus 여기까지의 과정이 'Style14_1_blend_start_course2.ai'으로 저장되어 있습니다.

4. 'Layer 6'을 선택하고 붙여넣기 합니다. 마우스 오른쪽 버튼을 눌러 Release Clipping Mask를 누르고 이미지에 X 표시가 뜬 개체만 삭제합니다.

5. 삭제했던 위치에 색이 없는 투명한 개체를 선택하고 면색을 검은색으로 설정합니다.

6. 'Layer 3'을 선택하고 지느러미 그라디언트 개체들을 모두 선택합니다. 이때, 블렌드로 만든 복잡한 개체가 아닌 아래쪽 개체를 선택해야 합니다. 복사한 뒤 'Layer 6'을 선택하고 붙여넣기 합니다. 그룹으로 만들고 지느러미 개체를 몸통으로 이동합니다.

7. 몸통과 지느러미를 모두 선택하고 패스파인더에서 모든 개체를 합칩니다. 면색은 검은색이며 [Effect]-[Blur]-[Gaussian Blur]를 클릭하고 10을 입력합니다. Ctrl+Shift+F10을 눌러 [Transparency] 패널을 열고 블렌드 모드를 Multiply, Opacity 값을 30%로 설정합니다.

8. [Layers] 패널을 열어 가장 위에 빈 레이어를 만들고 나머지는 모두 잠급니다. Pen Tool로 흰색, 선 두께는 5pt로 설정하고 그림과 같이 곡선을 그립니다.

9. Width Tool로 선 두께를 조절합니다. 일부는 굵고 일부는 얇게 만들어 자연스럽게 변화하는 물결을 표현합니다.
* Tip: Width Tool을 사용할 때 Alt 키를 누른 채로 선 두께를 조절하면 양쪽의 두께를 다르게 조절할 수 있습니다.

10. 모든 개체를 선택하고 [Object]-[Expand Appearance]로 확장합니다. 패스파인더 패널에서 Unite를 누르면 모두 합쳐진 면이 됩니다.

11. [Object]-[Path]-[Simplify]를 실행합니다. 화면에 조절창이 나타나면 슬라이더를 왼쪽으로 조절하여 점의 개수와 복잡도를 줄입니다.

12. 뾰족한 부분을 찾아 확대하고, Direct Selection Tool로 핸들을 조절하거나 Smooth Tool(매끄럽게 도구) 등을 활용하여 둥근 모양으로 만듭니다.
* plus 여기까지의 과정이 'Style14_1_blend_start_course3.ai'으로 저장되어 있습니다.

13. 합쳐진 개체를 Ctrl+8을 눌러 결합합니다. [Gradient] 패널에서 흰색 색상 점을 6개로 만듭니다. 2번째, 4번째, 6번째 Opacity 값은 0%, 각도는 90도로 입력합니다. 위치는 랜덤으로 설정합니다.

14. [Effect]-[Blur]-[Gaussian Blur]를 실행하고 값을 15px 입력하고 OK를 누릅니다. [Transparency] 패널을 열고 Opacity 값을 40%로 입력합니다.

15. [Object]-[Path]-[Offset Path]를 실행하고 값을 -2mm 입력한 뒤 OK를 누릅니다.

16. 다시 [Offset Path]를 실행하고 값을 -1.5mm 입력한 뒤 OK를 누릅니다. 안쪽으로 점점 작아지면서 중첩되는 효과를 만듭니다.
* Tip: 만약 흰색이 너무 강하게 표현된다면 Opacity 값을 원하는 만큼 낮춰줍니다.

17. 210x280mm 사각형을 만들어 전체 선택한 뒤 Ctrl+7을 눌러 클리핑 마스크를 적용합니다.

Step 8. 그라디언트 메쉬로 물방울 만들기

1. 물고기 입 주변에 흰색 정원을 그립니다. Mesh Tool(망 도구)을 선택하고 중앙을 클릭하여 메쉬를 만듭니다.

2. 바로 [Transparency] 패널을 열고 Opacity 값을 30%로 지정하여 해당 점의 투명도를 낮춥니다.

3. Mesh Tool로 위쪽에 점을 추가하고 Opacity 값을 0%로 지정하여 해당 점을 투명하게 만듭니다.

4. 만든 개체를 다양한 크기와 각도로 여러 개 복사합니다. 투명도 값을 40~80%로 다양하게 적용하여 기포를 자연스럽게 표현해 줍니다.

5. 전체적으로 개체의 배치와 블렌드, 색상, 그라디언트 등 어색한 부분이 없는지 확인하고 작업을 마무리합니다.

10. 메쉬(Mesh)를 활용한 인물 일러스트 제작

🚀 스텝별 제작 과정

Step 1. 캐릭터 얼굴 표현하기

* 개체의 기본 형태를 만들고 메쉬 작업을 시작함. 처음 개체를 그릴 때, 핸들의 길이가 너무 길지 않도록 주의해야 합니다.

1. [File]-[Open] → 'Style15_1_Mesh_start.ai' 파일 열기
* plus 대지 영역에 있는 그라디언트 개체는 머리카락이나 피부색 등을 표현할 때 사용함

2. [Pen Tool] → 얼굴의 절반만 막힌 도형으로 그리기 → 면색: #fdcfad
* Tip: 망 도구는 핸들 길이의 영향을 받기 때문에 처음 얼굴을 그릴 때, 핸들을 너무 길게 뽑아서 만들지 않고 각 점마다 핸들의 길이가 비슷하도록 작업하는 것이 좋음

3. [Layers] (F7) → Layer 2를 '작업'으로 이름 변경 → 'Sketch' 레이어 아래로 이동

4. 개체 선택 → [Mesh Tool] → 볼 왼쪽 부분, 입 아래쪽 부분을 클릭하여 메쉬 추가

5. 이마 가운데 부분과 눈썹 아래쪽도 클릭하여 추가 → [Layers] → 'Sketch' 레이어 눈 끄기

6. [Lasso Tool] → 가장 위쪽의 점 5개 선택 → 면색: #925133
- 두 번째 표시된 점들: #f6ab7c, 세 번째 표시된 점 1개: #f6b48b
* plus 예제와 똑같이 만들지 않아도 되므로 자유롭게 디자인함

7. 첫 번째 점들: #ecad83, 두 번째 점: #ffbd8e, 마지막 점: #ffdaaf

8. 첫 번째 점들: #f9e2c7, 두 번째 점: #ffdeb2, 마지막 점: #ffbe92

9. 첫 번째 점: #fec299 → 개체를 전부 선택 → [Reflect Tool]로 복제
* plus 만약 어색하게 느껴지는 부위의 색상이 있다면 해당 점을 선택하고 [Eyedropper Tool]로 주변 색을 추출하여 자연스럽게 변경함

Step 2. 방사형 메쉬로 귀 만들기

1. 12mm 정원 그리기 → 방사형 그라디언트 적용 (왼쪽 색상점: #fd7b4d, 오른쪽 색상점: #fce2c2)

2. [Object]-[Expand] → Gradient Mesh 선택 → [OK] → 그룹 해제

3. 다시 Ungroup 눌러 그룹 해제 → Release Clipping Mask로 클리핑 마스크 해제

4. 방사형 메쉬 개체 생성 완료 → 개체의 안쪽에 투명한 원을 찾아 삭제

5. 귀 모양을 만들어 배치 → 화면 확대 → [Mesh Tool]로 외곽 쪽에 점 추가

6. 귀 모양에 어울리도록 점들을 이동 → 어두워지는 부분 색상: #fb8656
- [Eyedropper Tool] 활용하여 주변 색상을 더 어둡거나 밝게 조절

7. [Layers] → 'Sketch' 레이어 눈 켜기 → 귀의 위치 맞추고 반대편은 반전 및 복사
- 귀 개체 2개, 얼굴 개체 2개를 각각 그룹으로 관리 → 모든 개체 선택 → Ctrl+2로 임시 잠금

💡 ESSENTIAL THEORY: Radial Gradient Mesh

그라디언트 메쉬 도구로 개체에 점을 찍으면, 일반적인 그물 형식의 메쉬가 만들어짐. 만약 중심에서 바깥쪽으로 퍼지는 방사형의 그라디언트를 만들려면 원형 그라디언트 개체를 Expand로 확장해야 함(확장 시 Gradient Mesh 선택). 방사형 메쉬는 CD, 레코드 레이블, 접시 등 원형 모양 개체에 주로 사용됨.

Step 3. 눈과 눈썹 표현하기

1. [Pen Tool] → 눈썹과 눈의 외곽 그리기 (눈썹 면색: #b14c36, 눈 면색: 흰색) → 아이라인 표현 부분에 검은색 선 그리기

2. 라인과 눈 개체 선택 → 패스파인더의 Divide 적용 → 그룹 해제 → 아이라인: 검은색, 눈: 흰색 설정

3. 스케치 크기에 맞춰 눈 안에 원 그리기 → 방사형 그라디언트 적용 (첫 번째 색: #f16144, 두 번째 색: #a02f1f(위치 85%), 세 번째 색: #000000(위치 100%)) → [Layers] → 'Sketch' 레이어 눈 끄기

4. 안쪽에 검은색 원과 흰색 원을 그려 눈동자 표현

5. 흰색 눈 개체 선택 → 선형 그라디언트를 90도로 적용 (왼쪽 색: #fcdec4, 오른쪽 색: 흰색) → 선색: #fccec0, Inside 정렬 적용

6. [Mesh Tool] → 눈썹 위쪽에 얇게 3줄, 아래쪽에 얇게 3줄 메쉬 추가

7. 위쪽 2줄: #bc694b (원래 눈썹 색보다 약간 밝게) / 아래쪽 2줄: #8e371c (약간 어두운 갈색)

8. 눈과 눈썹 개체 모두 선택 → 얼굴 가운데 기준 반전 및 복사 → 그룹 만들기

Step 4. 코와 입 표현하기

1. [Layers] (F7) → 'Sketch' 레이어 눈 켜기 → 8x6mm 원 그리기 (면색: #fdcdaa) → 'Sketch' 레이어 눈 끄기 → [Mesh Tool]로 점 찍어 메쉬 생성

2. 첫 번째 표시된 점들: #fbd5b8 / 두 번째 점: #f9dec7

3. 첫 번째 표시된 점들: #eeb792 / 두 번째 점: #eaae86

4. 'Sketch' 레이어 눈 켜기 → 7×10mm 원(선 두께 1.5pt) 생성 → [Object]-[Path]-[Outline Stroke]로 선을 면으로 확장

5. 입 가장자리 표현용 작은 원 생성 → 약간 회전시켜 배치 → 가운데 기준 반전 및 복사

6. 개체 3개 모두 선택 → [Shape Builder Tool] → Alt+클릭하여 불필요한 부분 삭제 후 나머지 병합 → 면색: #fec7a1

7. [Mesh Tool] → 개체 정중앙 살짝 아랫부분 클릭하여 메쉬 추가 → 원 안쪽에도 각각 하나씩 점 추가 → 안쪽 점 색상만 #db976a로 지정

8. 큰 원과 작은 원 배치 → [Pathfinder] 또는 [Shape Builder Tool] 활용하여 가장 아랫부분만 남기기

9. 입술 아래쪽 그림자 표현 → 면색: #f9ba8f → [Layers] → 'Sketch' 레이어 눈 끄기

Step 5. 머리카락 표현하기

1. 가장 앞쪽의 앞머리 부분을 스케치 라인 따라 2개의 개체로 그리기 (앞 큰 개체: #b14c36, 뒤 작은 개체: #490f0a)
- 작은 개체 선택 → Ctrl+2로 임시 잠금 → 'Sketch' 레이어 눈 끄기

2. 왼쪽 끝 뾰족한 부분 점 선택 → [Anchor Point Tool]로 핸들 조절하여 자연스럽게 다듬기
* Tip: 핸들이 한쪽만 있을 때보다 양쪽으로 있을 때 메쉬가 더 정리된 모양으로 생성됨

3. [Mesh Tool] → 왼쪽 아랫부분과 윗부분 클릭하여 메쉬 생성

4. 안쪽 부분에도 점 추가
* Tip: 메쉬는 점을 찍을 때 주변 점의 핸들 영향을 다르게 받기 때문에, 여러 번 반복 작업을 통해 어떤 순서가 가장 좋은 방법일지 고민하며 작업해야 함

5. 첫 번째 점들: #732412 / 두 번째 점들: #5b1a0e

6. 첫 번째 점들: #833221 / 두 번째 점들: #923a28
* plus [Eyedropper Tool]로 주변을 클릭하여 어색하지 않게 다듬는 것도 좋음

7. 두께감 표현용 메쉬 추가 → 일부를 선택하여 조금 더 밝은 색상 적용

8. 앞 큰 개체 잠금 설정 → 안쪽 개체 잠금 해제 → 안쪽에 음영 지는 부분 메쉬 추가
- [Lasso Tool]로 원하는 점 선택 → [Eyedropper Tool]로 주변 색 복사하여 톤 다듬기

9. Ctrl+Alt+2로 잠긴 개체 해제 → 큰 머리카락 개체 복사 → Ctrl+[로 뒤로 보내기
- 얼굴과 귀 개체 동시 선택 → Ctrl+Shift+[로 맨 뒤로 배열

10. 'Sketch' 레이어 눈 켜기 → 복사한 머리카락을 스케치 위치에 배치
- [Warp Tool] 더블클릭 → Intensity: 10%, Simplify 체크 해제 → [OK]

11. 머리 개체 클릭-드래그하여 스케치 라인과 비슷하게 왜곡
* Tip: Warp~Wrinkle 도구의 브러쉬 사이즈 조절: Alt+Shift 누른 채로 대각선 방향 드래그

12. 옆 부분 머리 개체 그리기 → 'Sketch' 레이어 눈 끄기 → 면색: #b14c36

13. 개체에 점 추가 → [Lasso Tool]로 오른쪽 외곽 부분의 점 모두 선택 → 면색: #44120e

14. 왼쪽은 밝고 오른쪽은 어둡게 연결되도록 색상 지정 작업

15. [Warp Tool]로 형태 다듬기 → 반대쪽 반전 및 복사

16. 개체 복사 → Ctrl+Shift+[로 가장 뒤로 보내기 → 'Sketch' 레이어 눈 켜기 → 이전과 같은 방법으로 [Warp Tool] 적용하여 형태 맞추기

17. 동일한 방법으로 뒤쪽 머리 개체 복사 및 형태 맞추기 반복 작업

18. 뒤쪽 머리 개체 일부에 메쉬 추가

19. [Lasso Tool]로 일부 점 선택하여 색상 변경 → [Eyedropper Tool]로 추출하여 자연스럽게 색이 연결되도록 작업
- 맨 뒤로 보내기 → 복사본 생성 → [Warp Tool]로 형태 다듬기

20. 전체 완성 후 자연스럽지 못한 부분 색상 조절 → 맨 뒤에 배경 사각형 생성 → 어울리는 그라디언트 적용하여 마무리

5. 특수 효과 (그림자/빗반사) 레시피

입체감과 고급스러움을 더해주는 자연스러운 그림자와 빗반사를 만드는 2가지 비법입니다.

방법 1. 그라디언트(Gradient) 그림자

색상 자체를 자연스럽게 빼면서 그림자를 만듭니다.

  1. 여러 개체라면 반드시 Ctrl+8 (컴파운드 패스) 적용
  2. G 누르고 Enter (그라디언트 90도 선형 적용)
  3. 양쪽 색상을 원본 도형과 동일하게 맞춥니다. (어색한 검정/흰색 X)
  4. 사라질 쪽 컬러를 선택하고 Opacity(투명도) 0% 적용
  5. 도형 전체 투명도를 40~60%로 낮춥니다.

방법 2. 오퍼시티 마스크 빗반사

마스크를 씌워 빛이 반사되는 듯한 효과를 냅니다.

  1. 원본 복사본 위에 '흰색~검정' 그라디언트 사각형 덮기
  2. 둘 다 선택 후 Transparency 패널 → Make Mask
  3. 마스크 썸네일(오른쪽) 클릭 후 G 눌러 방향 조절
  4. 완전한 투명을 위해 검정색 코드를 #000000으로 맞춥니다.
  5. 썸네일 왼쪽(원본) 클릭 후 도형 전체 투명도 40~60% 조절
