일러스트레이터 핵심 참고서

강의 중 헷갈리는 기능과 실전 로고 제작법 가이드

V 1.0

1. 기본 환경 설정 (작업 전 필수)

환경 설정 창 열기 Ctrl + K

  • 초기화: 하단의 [Reset Preferences] 클릭 → OK → 재시작 (작업 이력만 지워지고 파일은 안전합니다)
  • 시작 화면 끄기: General → [Show the Home Screen...] 체크 해제
  • 폰트 이름 한글로 보기: Type → [Show Font Names in English] 체크 해제
눈 피로도 감소

User Interface → Brightness를 어두운 테마로 설정

자동 저장 (매우 중요)

File Handling → Automatically Save... 체크 후 1~2분 간격으로 설정

작업 화면(패널) 및 툴바 정리

  • 상단 Window → Workspace → Essentials Classic 선택
    (패널이 꼬였을 때는 Reset Essentials Classic 클릭)
  • 선택할 때마다 뜨는 성가신 바 끄기: 상단 Window → Contextual Task Bar 체크 해제

2. 화면 조작 및 새 문서 설정

새 문서 만들기 Ctrl + N

※ 주의: 크기 숫자를 적기 전에 반드시 '단위(mm)'부터 맞추세요!

  • 크기: 500 x 500 mm (정사각형 로고 기준)
  • 컬러 모드: 웹(모니터)용은 RGB / 인쇄용은 CMYK
  • 해상도(Raster Effects): 웹/일반 작업은 72ppi (크면 컴퓨터가 느려집니다)

기본 선택 / 그리기

  • V 선택 도구 (검은 화살표)
  • A 직접 선택 (점 수정 / 흰 화살표)
  • M 사각형 도구
  • L 원형 도구

화면 제어 / 색상 도구

  • Space + 드래그 화면 이동
  • Alt + 휠 화면 확대 / 축소
  • D 기본 색상 (흰 면, 검은 선)
  • / 색상 채우기 없음 (투명)
Ctrl + S 저장
Ctrl + Z 되돌리기
Ctrl + Shift + Z 다시 실행

3. 로고 제작에 꼭 필요한 기능

실무 컬러 패널 세팅 (HSB)

Color 패널을 꺼내 Properties 위에 얹으세요.

패널 우측 상단 줄 3개(≡) 클릭 → HSB 선택
(색상, 채도, 명도로 조절하여 색 맞추기가 훨씬 쉽습니다)

모양 굳히기 (Expand)

비율을 조절한 도형의 현재 상태를 완전히 굳힙니다.

Object → Expand

※ 주의: 선(Stroke) 색상은 없애고 진행하세요.

정확한 수치로 이동 및 복제 (Move)

도형 선택 후 V(선택 도구) 상태에서 Enter 키를 누릅니다.

  • Horizontal: 가로 이동 (오른쪽 양수, 왼쪽 음수)
  • Vertical: 세로 이동 (아래쪽 양수, 위쪽 음수)
Preview 미리보기 Copy 복제 Ctrl + D 방금 한 행동 반복

도형 변형 (Transform)

S 크기 조절 (Scale) 비율 무시는 Non-Uniform
O 반전/대칭 (Reflect)
R 회전 (Rotate)

기준점 변경: 도구 선택 상태에서 Alt + 화면 클릭

일정한 크기로 확대/축소 (Offset Path)

원의 반지름 등을 기준으로 정확히 커지거나 작아집니다.

Object → Path → Offset Path

예) 지름 40mm(반지름 20mm) 원에 Offset 값 '-10mm'를 주면 반지름 10mm인 작은 원이 생성됩니다.

스포이트 도구 (Eyedropper) I

색상 넣어줄 도형 선택 후 → 스포이트로 훔쳐올 도형 클릭

  • 클릭: 면색 + 선 속성 모두 가져옴
  • Alt + 클릭: 반대로 적용 (색을 뿜어냄)
  • Shift + 클릭: 선 두께 무시, 오직 색상만 가져옴

4. 고급 도형 제어

도형 자르고 합치기 (Shape Builder) Shift + M

※ 작업할 도형들을 '모두 선택'하는 것이 1순위입니다!

  1. 분리: 그냥 클릭
  2. 합치기: 훑듯이 클릭 + 드래그
  3. 지우기(빼기): Alt 누른 상태로 클릭 / 드래그

작업 위치를 정확히 보려면 Ctrl + Y (윤곽선 모드)를 켜서 모서리를 맞추세요.

단순 묶기: 그룹 (Group)

도형을 한 묶음처럼 취급. 각각의 개체는 살아있습니다.

Ctrl + G

물리적 결합: 컴파운드 패스

여러 도형을 완전한 하나의 도형처럼 취급. (그라디언트 필수)

Ctrl + 8

자연스러운 그림자/빗반사 만드는 2가지 비법

방법 1. 그라디언트(Gradient) 그림자

  1. 여러 개체라면 반드시 Ctrl+8 (컴파운드 패스) 적용
  2. G 누르고 Enter (그라디언트 90도 선형 적용)
  3. 양쪽 색상을 원본 도형과 동일하게 맞춥니다. (어색한 검정/흰색 X)
  4. 사라질 쪽 컬러를 선택하고 Opacity(투명도) 0% 적용
  5. 도형 전체 투명도를 40~60%로 낮춥니다.

방법 2. 오퍼시티 마스크 빗반사

  1. 원본 복사본 위에 '흰색~검정' 그라디언트 사각형 덮기
  2. 둘 다 선택 후 Transparency 패널 → Make Mask
  3. 마스크 썸네일(오른쪽) 클릭 후 G 눌러 방향 조절
  4. 완전한 투명을 위해 검정색 코드를 #000000으로 맞춥니다.
  5. 썸네일 왼쪽(원본) 클릭 후 도형 전체 투명도 40~60% 조절

5. 실전 예제: 로고 제작 레시피

변호사 로펌 로고 (신뢰감 컨셉)

추천 컬러: 남색, 청록색, 차분한 고동색 등 신뢰감을 주는 색상

1. 기본 기둥 세우기: 사각형(M) 5x43mm 생성 → HSB 패널로 남색 적용

2. 복사하기: V + Enter (가로 7.5mm / 세로 0) Copy 클릭 → Ctrl + D 로 3개 완성

3. 사선으로 자르기: 35x35mm 정사각형 생성 → 45도 회전 → S + Enter (세로 60% 납작하게) → Object > Expand 굳히기 → Ctrl+Y (윤곽선 모드)로 모서리 맞추기 → Shift+M 누르고 Alt+드래그하여 튀어나온 부분 삭제.

4. 그룹 후 반전(대칭): 세 개 도형 잡고 Ctrl+G → O(반전) 누르고 우측 앵커포인트에 Alt+클릭 → Vertical(세로) Copy 클릭.

5. 톤 맞추기: 반전된 도형은 HSB 패널에서 S(채도)는 낮추고 B(명도)는 높여 파스텔 톤으로 변경.

6. 그림자 마감: 상하 반전 → 도형 합치기 → 컴파운드 패스(Ctrl+8) → 그라디언트 그림자 적용.

태극 로고 (공공기관, 자격증 단골)

1. 기본 원 생성: 원형 도구(L) 40x40mm 생성 → 기본색(D) 적용 → 면색 제거(/)

2. 정확한 절반 크기 원 만들기: 원 선택 → Object > Path > Offset Path (값: -10mm) 입력.
(지름 40mm의 반지름은 20mm. -10mm를 주면 반지름 10mm인 정확한 반쪽짜리 원이 생깁니다.)

3. 정렬 및 복사: 큰 원, 작은 원 모두 선택 → Align(정렬) 패널 '왼쪽 정렬' → 작은 원 복사 (Ctrl+C, Ctrl+F 제자리 붙여넣기)

4. 우측 정렬: 큰 원과 방금 복사한 작은 원을 같이 선택 → Align 패널 '오른쪽 정렬'

※ 눈대중으로 맞추면 안 됩니다! 반드시 Offset Path와 Align 패널을 사용하세요.

6. 폰트(글꼴) 실무 사용 가이드

상업적 이용 가능한 무료 폰트

상세페이지, 배너, 포트폴리오 등에는 반드시 상업적 무료 폰트를 사용하세요.

한글 폰트 눈누 (Noonnu)
영문 폰트 구글 폰트 (Google Fonts)

※ 주의: '다폰트(Dafont)' 등의 사이트는 개인용 무료가 많으므로 라이선스 확인에 각별히 유의해야 합니다.

실무 폰트 설정 공식

글자 가로/세로 비율 장평

건들지 말기 (100%)

글자 사이 간격 자간

음수값 (-25 ~ -50 등)

줄 사이 간격 행간

폰트 사이즈 × 1.2~1.6배

가독성이 중요한 본문

  • 웹(화면)용: 고딕 계열 추천
  • 인쇄물용: 명조 계열 추천

고딕 추천: Noto Sans KR, Pretendard, Wanted Sans

명조 추천: 부크크 명조, 리디 바탕, 고운 바탕

꾸미기 & 타이틀

  • 감성적인 디자인: 장평이 긴 폰트나 세리프 추천
  • 활동적인 디자인: 장평이 넓고 짧은 폰트 추천

포인트용 추천: 페이퍼로지, 에이투지체

TTF (일반 문서)

워드, PPT 등 일반 문서 작업용

Windows OS

OTF (그래픽 작업)

일러스트, 포토샵 등 그래픽 작업용

Mac OS
