🎨 일러스트레이터 핵심 참고서

강의 중 헷갈리는 기능과 단축키를 바로 확인하세요.

⚙️ 1. 기본 환경 설정

환경 설정 창 열기: Ctrl + K

  • 초기화: 환경 설정 창 하단의 Reset Preferences 클릭 → OK → 프로그램 재시작
  • 시작 화면 끄기: General → Show the Home Screen... (체크 해제)
  • 폰트 이름 한글로 보기: Type → Show Font Names in English (체크 해제)

작업 화면(패널) 초기화

상단 메뉴바에서 Window → Workspace → Essentials Classic 선택
(패널이 꼬였을 때는 Reset Essentials Classic을 누르세요.)

⌨️ 2. 화면 조작 및 필수 단축키

기본 도구

  • V 선택 도구 (검은 화살표)
  • A 직접 선택 도구 (흰 화살표)
  • M 사각형 도구

화면 제어

  • Space + 마우스 드래그 화면 이동
  • Alt + 마우스 휠 화면 확대 / 축소
  • Ctrl + Y 윤곽선 모드 보기

파일 관리 및 되돌리기

Ctrl + N 새 문서 생성
Ctrl + S 저장하기
Ctrl + Z 되돌리기 (실수했을 때)
Ctrl + Shift + Z 되돌린 것 다시 실행

🛠️ 3. 꼭 알아야 할 필수 기능

줄자 켜기 / 단위 맞추기

Ctrl + R 을 눌러 줄자를 켭니다.

줄자 위에서 마우스 우클릭 후, 만든 종이(대지)의 단위(예: mm, px)와 동일하게 설정하세요.

제자리(바로 앞)에 복사하기

Ctrl + C 후 Ctrl + F

선택한 도형과 완벽히 똑같은 위치의 바로 앞에 복사본을 붙여넣습니다.

정확한 수치로 이동 및 복제 (Move)

도형 선택 후 V(선택 도구) 상태에서 Enter 키를 누릅니다.

  • Horizontal (가로): 양수는 오른쪽, 음수는 왼쪽으로 이동
  • Vertical (세로): 양수는 아래로, 음수는 위로 이동
  • Angle (각도): 가로/세로 값을 입력하면 각도 값이 자동으로 계산되어 대각선으로 이동합니다.
Preview: 이동할 위치 미리보기 Copy: 해당 위치로 복제 OK: 그냥 이동

💡 꿀팁! 방금 한 행동(이동/복제)을 반복하려면 Ctrl + D 를 누르세요.

도형 변형 (Transform)

S 크기 조절 (Scale)
O 반전/대칭 (Reflect)
R 회전 (Rotate)

기준점 변경하기: 도구를 선택한 상태에서 Alt 키를 누른 채 화면의 다른 곳을 클릭하면, 그곳이 새로운 기준점이 되며 설정 창이 뜹니다.

스포이트 도구 (Eyedropper Tool) I

▶ 기본 사용 방법

1단계 색상을 넣어줄 도형 선택 ➔ 2단계 스포이트로 색상을 훔쳐올 도형 클릭

* 기본 클릭 시 면색과 선의 속성(두께, 색상 등)을 모두 가져옵니다.

  • Alt + 클릭 반대로 적용! (색상을 훔쳐올 도형 선택 후, Alt 누르고 넣어줄 도형 클릭)
  • Shift + 클릭 선 두께 등의 속성은 무시하고 오직 색상만 가져옵니다.

✨ 4. 마법 같은 고급 기능

도형 구성 도구 (Shape Builder) Shift + M

도형을 쉽게 분리, 합치거나 자를 수 있습니다.

  1. 합치거나 자를 도형들을 모두 선택합니다.
  2. 단축키 Shift + M 을 누릅니다.
  3. 클릭: 교차된 면을 각각 분리합니다.
  4. 클릭 + 드래그: 훑고 지나간 도형들을 하나로 합칩니다.
  5. Alt + 클릭 / 드래그: 원치 않는 부분을 지우거나 뺍니다.

그룹 (Group)

여러 개의 도형을 한 묶음처럼 취급합니다. (언제든 다시 풀 수 있음)

Ctrl + G

컴파운드 패스 (Compound Path)

여러 개의 도형을 물리적으로 완전한 하나의 도형처럼 취급합니다.

Ctrl + 8

불투명도 마스크 (Opacity Mask)

무채색(그라디언트 등) 도형을 위에 얹어, 아래 도형을 자연스럽게 가리거나 보이게 하는 기능입니다.

1. 적용할 도형들을 모두 선택합니다. (위에 덮은 무채색 도형 + 아래 그림)

2. 투명도(Transparency) 패널에서 Make Mask 버튼을 누릅니다.

  • 검정색: 이미지를 가립니다.
  • 흰색: 이미지를 보여줍니다.

📝 5. 폰트(글꼴) 실무 사용 가이드

상업적 이용 가능한 무료 폰트

상세페이지, 배너, 카드뉴스, 포트폴리오 등에는 반드시 상업적 이용이 허락된 무료 폰트를 사용하세요.

🇰🇷 한글 폰트 눈누 (Noonnu)
🔤 영문(외국어) 폰트 구글 폰트 (Google Fonts)

* 주의: '다폰트(Dafont)' 등의 사이트는 개인용 무료가 많으므로 라이선스 확인에 각별히 유의해야 합니다.

가독성이 중요한 본문

굵기(Weight)가 다양한 폰트 패밀리를 추천합니다.

  • 웹(화면)용: 고딕 계열 (산세리프) 추천
  • 인쇄물용: 명조 계열 (세리프) 추천
[추천 한글 폰트]
  • 고딕: Noto Sans KR(구글), Pretendard(눈누), Wanted Sans(눈누)
  • 명조: 부크크 명조, 리디 바탕, 고운 바탕 (모두 눈누)

꾸미기 & 타이틀

분위기와 목적에 맞는 개성 있는 폰트를 선택하세요.

  • 감성적인 디자인:
    장평이 세로로 긴 폰트나 부드러운 명조(세리프) 추천
  • 활동적이고 통통 튀는 디자인:
    장평이 가로로 넓고 세로로 짧은 폰트 추천
[포인트 + 본문용 추천]

페이퍼로지, 에이투지체 (눈누)

📐 실무 폰트 설정 공식

글자 가로/세로 비율 장평

건들지 말기 (100%)

글자 사이 간격 자간

음수값 (-25 ~ -50 등)

줄 사이 간격 행간

폰트 사이즈 × 1.2~1.6배

TTF (TrueType Font)

일반 문서 작업 (워드, 엑셀, PPT 등)

주로 Windows OS에서 사용

OTF (OpenType Font)

그래픽 작업 (일러스트, 포토샵 등)

주로 Mac OS에서 사용 (고해상도)
